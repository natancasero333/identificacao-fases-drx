# Identificação de Fases por DRX

## 📌 Descrição

Este projeto realiza a identificação de fases cristalinas a partir de difratogramas de raios X (DRX), comparando dados experimentais com padrões simulados a partir de arquivos CIF.

---

## ⚙️ Metodologia

1. Leitura do difratograma experimental
2. Normalização da intensidade
3. Comparação entre padrões
4. Ranking das fases mais prováveis
5. Plotagem de gráfico para comparação
6. 
7. 

---

## 📊 Resultado

O código retorna um score de comparação entre os dados experimentais e os dos CIF carregados.
